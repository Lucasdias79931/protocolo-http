# Análise de Requisições HTTP com FastAPI

Este projeto tem como objetivo analisar requisições HTTP utilizando o **FastAPI** como objeto de estudo, explorando conceitos de backend, comunicação cliente-servidor e containerização com Docker.

---

##  Tecnologias utilizadas

- Flask

---


---

##  Como executar o projeto

###  Usando Docker Compose

```bash
    $ docker compose up --build 

```

## como executar sem docker

```bash
    $ gunicorn --workers=4 src.app:application

```
